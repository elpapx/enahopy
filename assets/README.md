# Assets

## Logo

The enahopy logo should be placed here as `enahopy_logo.png`.

**Requirements:**
- Format: PNG
- Recommended size: 800x200px or similar (4:1 aspect ratio)
- Transparent background preferred
- Should match the brand colors (brown/terracotta tones with textile pattern)

## Adding the logo

1. Save your logo file as `enahopy_logo.png` in this directory
2. Ensure it's optimized for web (< 100KB recommended)
3. Commit and push to repository

The README.md references this logo at: `assets/enahopy_logo.png`
